# vuepoint

## Project setup
```
yarn install
yarn bootstrap
```

### Compiles storybook and hot-reloads for development
```
yarn serve
```

### Compiles and minifies storybook for deployment
```
yarn storybook:build
```

### Compiles and minifies packages for production
```
yarn build
```

### Run unit tests
```
yarn test:unit
```

### Lints and fixes files
```
yarn run lint
```
